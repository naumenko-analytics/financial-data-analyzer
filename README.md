# Financial Data Analyzer
### Naumenko Analytics LLC

## Overview
An interactive financial data analysis tool built with Python and Streamlit. 
This application allows users to upload any financial dataset or connect to 
live stock market data to generate automated analysis, visualizations, and insights.

## Features
- Upload CSV/Excel files for instant data analysis
- Live stock market data via Yahoo Finance API
- Interactive charts and visualizations
- Automated statistical analysis and insights
- Downloadable reports

## Technologies Used
- Python 3.14
- Streamlit
- Pandas
- Plotly
- yfinance
- scikit-learn
